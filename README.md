# ⛽ Fuel Tracker / Suivi d'essence

Personal fuel expense tracker. Bilingual (FR/EN), localStorage persistence, station breakdown, period filtering.

**Live:** [https://frankmartymcfly.github.io/fuel-tracker/](https://frankmartymcfly.github.io/fuel-tracker/)

## Stack

- React 18 + Vite
- Recharts (area chart)
- Lucide React (icons)
- GitHub Pages (deploy via Actions)

## Dev

```bash
npm install
npm run dev
```

## Deploy

Push to `main` → GitHub Actions builds and deploys automatically.

Requires: **Settings → Pages → Source → GitHub Actions**
